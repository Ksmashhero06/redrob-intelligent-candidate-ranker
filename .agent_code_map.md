# Project Architecture Map

## Root Directory

The project is an AI Studio application named "Redrob Intelligent Candidate Ranker", moved to the root level `D:\primus\`.

### src/

Main source directory containing the frontend application (Vite + React + TypeScript).

* **App.tsx**: The main application entry point containing layout, state for tabs (Upload/Results), and API integration with Gemini.
* **main.tsx**: Standard React/Vite mount script.
* **index.css**: Root-level CSS styling.
* **types.ts**: Common interfaces/types for candidates, parsing, and ranking.

### src/components/

* **UploadTab.tsx**: Component handling file upload (PDF/TXT resumes), parsing status, and the job description inputs.
* **ResultsTab.tsx**: Component displaying the ranked results, comparison tables, and analytics.

### src/utils/

* **parser.ts**: Helper utilities for parsing resume text and structured responses.
* **sampleData.ts**: Hardcoded sample candidates and mock data for UI fallback or demonstration.
